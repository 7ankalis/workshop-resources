# Workshop Resources

This repository contains materials and resources for the workshops I lead or will lead as part of **Securinets ENIT**.  
It includes slides, notes, scripts, demos, and exercises covering various cybersecurity and technical topics.

## Structure

Each directory corresponds to a specific workshop and may include:
- Slides or presentation material  
- Code samples or demos  
- Exercises and solutions  
- References or documentation
